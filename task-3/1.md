# Вопрос 1 (Session 5: Datalog) — RA для `r` и `s`

Дано (экстенсиональные):
- `t(Y)`
- `u(V,W)`
- `v(W,X)`

Интенсивиональные:
- `s(X,Y) :- u(V,W), v(W,X), v(W,Y), not(X=Y).`
- `r(X,Y) :- s(X,X), t(Y), not(s(Y,Y)).`

## Шаг 1: выразим `s(X,Y)` через `u` и `v`

Пусть

- `U(W) := π_W(u(V,W))`
- `A(W,X) := U(W) ⋈ v(W,X)`

Тогда все пары, связанные через один и тот же `W`, получаются из двух копий `A`:

`S0(X,Y) := π_{X,Y}( A ⋈_{A.W = B.W} ρ_{B}(A) )`

и условие `not(X=Y)` (т.е. `X≠Y`) задаём выборкой:

`s(X,Y) := σ_{X≠Y}(S0(X,Y))`

## Шаг 2: выразим `r(X,Y)` через `s`

Определим диагональ:

`Diag(X) := π_X( σ_{X=Y}( s(X,Y) ) )`

и множества значений `Y`, для которых `s(Y,Y)` истинно:

`DiagVals(Y) := π_Y( σ_{X=Y}( s(X,Y) ) )`

Тогда из условия `not(s(Y,Y))` получаем:

`r(X,Y) := Diag(X) × ( t(Y) − DiagVals(Y) )`

## Уточнение по данным правилам

Из правила для `s` следует `not(X=Y)`, значит `s(Y,Y) = ∅`. Поэтому `DiagVals(Y) = ∅`, а также `Diag(X)=∅` (диагональные кортежи в `s` недостижимы). Следовательно:

`r = ∅`.

# Ответ на вопрос 1

Даны экстенсиональные предикаты `parent(X,Y)` (X — родитель Y) и `person(X,G)` (у X пол/гендер `G`).
Нужно определить интенсивиональные `aunt`, `uncle`, `ancestor`.

Ниже пример Datalog-программы.

## Вспомогательные правила

```prolog
female(X) :- person(X, female).
male(X)   :- person(X, male).

% братья/сёстры: X и Y имеют общего родителя и не равны
sibling(X,Y) :- parent(Z,X), parent(Z,Y), not(X=Y).
```

## uncle и aunt

```prolog
% uncle(X,Y): X — мужской родственник-поддержка для родителя Y (мужской sibling родителя Y)
uncle(X,Y) :- male(X), parent(Z,Y), sibling(X,Z).

% aunt(X,Y): X — женский sibling родителя Y
aunt(X,Y)  :- female(X), parent(Z,Y), sibling(X,Z).
```

## ancestor

```prolog
% ancestor(X,Y): X — прямой или непрямой предок Y
ancestor(X,Y) :- parent(X,Y).
ancestor(X,Y) :- parent(X,Z), ancestor(Z,Y).
```

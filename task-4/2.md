# Вопрос 2 (Session 6: OLAP) — числа кортежей/ячеек

Дано: `Sales(Product,Month,Store,Amount)`
- Products: 5 (`P1..P5`)
- Months: 12
- Stores: 3 (`S1..S3`)

## 2(a) Dense setting

### i. Сколько кортежей в relation?

Для каждой комбинации `(product p, month m, store s)` есть ровно один кортеж с ненулевой суммой.

`5 * 12 * 3 = 180`

### ii. Сколько ячеек содержит data cube по измерениям Product, Month, Store?

Для одного численного измерения (мера `Amount`) число “кубиковых ячеек” равно:

`(|P|+1) * (|M|+1) * (|S|+1) = (5+1)*(12+1)*(3+1) = 6*13*4 = 312`

## 2(b) Sparse setting — сколько не пустых ячеек

Список ненулевых фактов (из условия):
- (P1, Jan, S1, a1)
- (P1, Jan, S2, a2)
- (P2, Feb, S2, a3)
- (P2, Feb, S3, a4)
- (P3, Jan, S1, a5)
- (P3, Feb, S1, a6)
- (P4, Feb, S1, a7)
- (P5, Jan, S3, a8)

Cube строится по измерениям `(Product, Month, Store)`, т.е. каждая ячейка соответствует одному из наборов “группировочных” измерений (остальные превращаются в NULL).

Считаем число непустых ячеек суммированием по всем наборам измерений:

### Все 3 измерения (P,M,S)

Фактов: `8` ⇒ непустых ячеек `8`.

### По 2 измерения (P,M), (P,S), (M,S)

1) `(P,M)` — различающиеся пары из фактов:
- (P1,Jan), (P2,Feb), (P3,Jan), (P3,Feb), (P4,Feb), (P5,Jan)

Итого `6`.

2) `(P,S)`:
- (P1,S1), (P1,S2)
- (P2,S2), (P2,S3)
- (P3,S1)
- (P4,S1)
- (P5,S3)

Итого `7`.

3) `(M,S)`:
- (Jan,S1), (Jan,S2), (Jan,S3)
- (Feb,S1), (Feb,S2), (Feb,S3)

Итого `6`.

### По 1 измерению

`Product`: 5 (P1..P5)  
`Month`: 2 (Jan, Feb)  
`Store`: 3 (S1..S3)

Итого: `5 + 2 + 3 = 10`.

### Общая агрегация (P,M,S → все NULL)

`1`

## Итого

`8 + (6+7+6) + (5+2+3) + 1 = 8 + 19 + 10 + 1 = 38`

Ответ: `38` непустых ячеек.
